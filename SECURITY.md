# Security Policy

## Reporting Vulnerabilities

If you discover a vulnerability in Apple's XNU kernel using this fuzzer:

1. **Do NOT disclose publicly** — follow responsible disclosure.
2. **Report to Apple** via their Security Bounty Program:
   https://security.apple.com/bounty/
3. Apple's bounty categories for kernel bugs:
   - Unauthorized access to sensitive data: $25K–$100K
   - Kernel code execution from app: $100K–$250K
   - Sandbox escape: $100K–$250K

## Reporting Issues in This Fuzzer

For bugs in the fuzzer itself (not in XNU), open a GitHub Issue on this
repository. Include:

- Steps to reproduce
- Expected vs actual behavior
- Platform and compiler version
- Build log if applicable

## Scope

This tool is intended for authorized security research only. Use it
responsibly and in compliance with all applicable laws and regulations.
